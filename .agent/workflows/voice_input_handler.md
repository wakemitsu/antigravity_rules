---
description: Automatic process upon voice input detection. Summarizes and repeats the user's input (instructions, questions, small talk, etc.) before proceeding.
---

// turbo-all

1. Analyze the voice input content to accurately understand the user's intent (e.g., work instructions, technical questions, progress reports, or casual conversation).
2. Briefly summarize and repeat the input in 1-2 lines to confirm understanding.
   - For instructions: Use phrases like "〜という指示として承りました" (I have received your instructions to...).
   - For questions: Use phrases like "〜というご質問ですね" (So your question is about...).
   - For small talk/reports: Use phrases like "〜というお話ですね" (So you're saying that...).
     **Note**: Always use natural Japanese for these responses.
3. Immediately follow the confirmation with the appropriate action (e.g., executing the task, providing an answer, or continuing the conversation).
